# 서울시 전동킥보드 불법 주정차 문제 해결 및 단속 최적화 프로젝트

## 프로젝트 소개
최근 공용 킥보드의 급격한 확산으로 인해 도시 내 보행 안전 저해, 보행로 점유, 단속 및 수거 비용 증가 등의 사회적 문제가 대두되고 있습니다. 특히 불법 주정차는 시민 안전과 행정 효율을 동시에 악화시키는 핵심 요인입니다. 본 프로젝트는 서울시 데이터를 기반으로 불법 주정차 문제를 분석하고, 단속 동선 최적화 및 주차 공간 입지 추천 솔루션을 제안하여 이러한 문제를 해결하고자 합니다.

## 사용 데이터
본 프로젝트는 서울 열린데이터 광장의 공공 데이터를 활용합니다.
* 서울시 전동킥보드 견인 현황 (https://data.seoul.go.kr/dataList/OA-21304/S/1/datasetView.do)
* 서울시 전동킥보드 주차구역 현황 (https://data.seoul.go.kr/dataList/OA-21710/S/1/datasetView.do)

## 주요 과제 및 해결 방안

### 1. 불법 주정차 단속 동선 최적화
한정된 인력과 시간 내에서 단속 실적(또는 예방 효과)을 최대화할 수 있는 순찰 경로를 계획합니다.

* 접근 방법: 구(District) 내에서 견인이 빈번한 지역의 우선순위를 도출하고, 이를 기반으로 최적의 단속 루트를 생성
* 사용 모델: KDE (Kernel Density Estimation), LightGBM
* 평가 지표: Top-k 적중률, 거리당 기대 단속 건수 (Reward/km), 경로 총 이동 거리

### 2. 주차 공간 추가 마련 (입지 추천)
불법 주정차를 최소화하고, 합법적인 주차 구역의 이용률을 최대화할 수 있는 신설 주차 구역을 추천합니다.

* 접근 방법: 견인 밀도, 기존 주차구역까지의 거리, 구역 포화도 등을 종합적으로 분석
* 사용 모델: XGBoost
* 평가 지표: 예상 위반 감소량 (%)
